# REST-Tester

## Requirements:
* Must have node & npm installed: https://nodejs.org/en/
* Use POSTMAN to test the API: https://www.getpostman.com/apps 

### To install:
```
    npm install
```

### To run:
```
    npm start
```
OR
```
    node server.js
```
